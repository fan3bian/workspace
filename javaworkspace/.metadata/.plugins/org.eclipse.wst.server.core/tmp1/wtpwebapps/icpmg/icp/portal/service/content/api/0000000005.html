stopVirtualMachine

关闭虚机

 

请求参数


名称

类型

是否必须

描述

id

String

是

要关闭的虚机ID

forced

bool

否

强制关闭虚机，即使消息发送失败，虚机也被置为关闭态，展示给用户的也为关闭态


响应参数


与重启虚机响应参数相同
