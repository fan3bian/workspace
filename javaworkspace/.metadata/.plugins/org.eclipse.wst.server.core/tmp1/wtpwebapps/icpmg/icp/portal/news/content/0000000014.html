【系列报道】“云潮”航母是怎样炼成的（三）

行业级云服务落地显王者风范

孔明说：“王者，携无虎之气，龙虎之子也。” 秦王嬴政的威严，唐太宗的亲和，成吉思汗的果敢，在历史的银幕上频频闪光

王者确实是应该有一种气质的。一种很强大的气魄与威严，一种发现趋势的预见与前瞻。它需要很多的积累与沉淀。

放之四海而皆准的道理，在IT界同样适用。

在波诡云谲的云计算竞争大潮中，谁有预见性和前瞻性，谁就占领了市场的制高点，拥有了行业的话语权。

从行业云到行业级云服务，浪潮在云服务领域不随波逐流，而是另辟蹊径，以高屋建瓴的姿态，引领云服务行业的新风潮。作为行业级云服务的领导者，浪潮低调而不张扬，却无法在云计算市场遮掩住王者的光芒。

从行业云到行业级云服务，浪潮总比市场快一步

“行业云”的概念，是由浪潮在2010年首次提出的。当时行业信息化对社会、对大众提供的信息服务几乎是空白，而云计算可以将海量数据的巨大潜在价值释放出来。行业云不仅可以实现行业内数据共享，也可以帮助行业数据拥有者将数据转换为服务，提升业务价值。

行业云真正落地的时候，前期也是以私有云的形式落地。这种“我花钱建我的，你花钱建你的”的私有云构建模式走了重复建设的路子，且不利于数据共享和数据应用。而行业用户采用公有云服务首先遇到的最大问题就是安全，行业云很多需要内外网隔离，而公有云开放统一的平台无法解决这一问题。至于定制化需求，公有云的高标准、高统一性，以及相关PaaS和SaaS方面软件和应用的缺乏也很难响应。此外最关键的，还有行业合规性问题。市场呼唤符合行业要求的行业云服务。

时隔4年，浪潮别出心裁，在国内首度推出行业级云服务，迈向以服务为交付模式的公有云市场。何为行业级云服务？简而言之，就是既关注个体云服务的解决，同时要关注和行业之间的数据共享。通过各类云端的服务搭配、组合及定制开发，满足行业客户不同规模、不同IT水平的多样化业务需求，任何行业都能在浪潮找到适合自己的高可信的云服务解决方案。

集产品供应、解决方案和运营服务“铁三角”能力于一身的浪潮，又在业内创新性地提出“一站式”服务交付的模式--假如浪潮提供的整体云服务好比是一辆品牌汽车，汽车的各个部件以及组装等的供应和服务由浪潮来统筹安排；同时，作为“汽车生产商”的浪潮可以利用自己的优势来生产汽车零部件的某一部件或完成生产过程中某一模块来提升整个汽车的性能；浪潮还可以组建服务队伍支持客户的业务运营。这种“一站式交付”战略以最终提供优质的服务为核心，而客户只需要专注于自身的核心业务。“千年的大道走成河”，没有多年软硬一体化的积淀，铸就不了浪潮铮铮的底气。

行业级云服务在全国攻城略地

当前，行业云的建设模式已出现“租建比”的拐点，各政企行业纷纷尝试通过购买服务的方式获取云计算资源。比如浪潮实施的山东警务云服务，在行业云服务的众多应用中以大数据的应用创新为特色。山东公安通过建设统一的警务云计算中心将警务应用系统全部迁移到“云”上。同时，利用大数据处理平台，整合公安内部数据、社会组织数据和互联网数据，打破了部门警种界限，可以深度挖掘并关联多种信息。

“警务云只是政务云的一个分支。”浪潮集团云服务事业部总经理颜亮在接受媒体采访时这样说。

济南政务云是全国首个整体服务外包的政务云。它以济南市公用信息平台为依托，由浪潮集团投资建设运营，建立了完善的运营管理体系，实现了以购代建，统一建设、统一监管，应用软件和技术支撑环境相分离。各部门按需求购买使用云中心服务资源，降低政府投入约30%。此外，浪潮还成立了专业的运营团队，保障济南市政府53个部门、300多项业务应用在云中心的高效运行。随后，北京市海淀区政府将该模式成功复制。

海南省省级政务云计算中心在云计算中心管理、资源使用申请、云服务价格制定、统一采购等方面做出了清晰的规定。而常德市政府建设的政府云，则将服务运营向上拓展至SaaS层面，制定了“建设云中心、整合数据资源、搭建公共服务平台”三步曲搭建智慧常德整体框架。哈尔滨、绵阳、长治等城市也在带动下采用了浪潮云服务，行业级云服务已在全国各大城市攻城略地。

聚焦行业级客户，向云服务商华丽转身

当前，全球云服务产业进入快速发展时期，并成为世界各国抢占新一轮经济和科技发展制高点的战略重点。据研究公司Gartner预测，全球公有云市场在2013年将增长18.5%至1310亿美元，去年则为1110亿美元。云计算使用案例也将逐渐从测试开发和案例部署，开始切实应用到部分行业的实践中。而亚太地区则成为公有云市场增长最快的地区之一，行业云服务不断兴起是推动公有云项目落地的有效动力。

各国政府积极通过政策引导、制定标准、投入资金等方式加快本国云服务的布局形成和产业发展。早在2010年底，奥巴马领导的美国政府率先提出“云优先”政策，明确规定各政府部门向云迁移的时间表，并对政策执行情况开展评估。美国政府也因此大量节省了IT成本，提升政府机构运作效率。英国、加拿大等也相继推出类似的政府“云优先”政策。

中国也开始布局鼓励政府购买服务的日程表。2013年，国务院办公厅发布96号文《关于政府向社会力量购买服务的指导意见》，鼓励各级政府向社会力量购买服务。

国家良好的政策环境为云服务产业的发展带来了机遇。在云服务领域长袖善舞的浪潮，加快向云服务战略华丽转身。浪潮已制定了主打“政务云”、培育“企业云”的战略规划，专注行业级云服务，持续提升行业级客户服务能力，发挥浪潮在技术方面的领先优势，大力发展浪潮云服务管理平台（ICP）、大数据处理平台(IOP)两大平台，为行业级云服务提供可靠支撑，布局北至黑龙江，南至海南的10大第四代数据中心，提供行业级云服务，在已达成的超过30个行业云战略合作基础上，大力推动济南 “政务云”模式的复制，进一步加强区域本地化建设，重点发展300家“云智联盟”合作伙伴，提升面向客户的本地化、专业化服务能力。
