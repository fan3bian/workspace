1月1日浪潮云服务客户中心开通公告

2015年1月1日，浪潮云服务客户中心正式上线开通，同时作为重要的服务渠道，客户中心微信公众平台与QQ服务平台也正式对外启用，这标志着浪潮云服务在着力提升服务质量与改善用户体验的发展道路上又迈出了坚实的一步。

浪潮云服务客户中心将为外部用户提供故障申告、技术支持、业务变更及咨询、投诉及建议四类服务。中心设有专人接听的400热线服务电话，接听人员设置为一线热线工程师3人，二线技术支持工程师2人，服务监督员1人。中心整体客服工作流程严格按照《服务台行为规范》与《浪潮云计算中心电话语言规范》标准执行，通过人员合理配置与规范性的管理方法做到全面、快速、高质量的满足用户服务需求。在今后的工作中，浪潮云服务用户如有售后服务问题建议直接拨打云服务客服400热线。服务台执行7*24在线服务，为用户提供全周期服务跟踪。

随着浪潮云服务客户中心的正式启用，我们将通过统一规范的对外服务口径、先进完备的技术支撑体系与快速及时的响应速度，承担起全国各区域的云用户服务工作，全力保障用户云服务业务的通畅，为全面树立浪潮云服务良好的品牌形象，进一步在全国范围内扩大浪潮云服务疆域奠定了坚实基础。同时随着云服务业务的不断扩大，我们也将持续完善扩充服务功能，提升服务水平，为浪潮云服务快速、稳定发展保驾护航。
