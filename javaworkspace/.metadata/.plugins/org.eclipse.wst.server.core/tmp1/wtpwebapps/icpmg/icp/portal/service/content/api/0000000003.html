listTemplates

获取模板

请求参数


名称

类型

是否必须

描述

templatefilter

String

是

featured:返回被标识为featured(精选)和public(公用)的模板；self:返回已经被注册的或者被调用者创建的模板；selfexecutable:与self意义相同，但是只返回可用来部署新虚机的模板;sharedexecutable:返回被其他用户授权调用者去部署虚机的模板；executable:被调用者拥有，或者可以用来部署虚机的公用模板；community:被标识为公用模板但没有标识精选的;all:所有的模板，只能被管理员所调用。

account

string

否

列出account下的模板，必须与domainid一同使用

domainid

string

否

 

hypervisor

string

否

限定返回特定的虚拟化平台的模板

id

string

否

模板id

isrecursive

bool

否

若为真，列出以domainid为父节点，到叶子节点所有domain下的模板。默认为false

keyword

string

否

 

listall

bool

否

若为真，则列出调用者被授权可以看到的所有模板；若为false，则只列出属于调用者的模板。默认为false
