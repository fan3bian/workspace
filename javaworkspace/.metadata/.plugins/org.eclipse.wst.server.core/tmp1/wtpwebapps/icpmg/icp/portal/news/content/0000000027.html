总理再次点赞浪潮云计算、大数据

3月6日上午，在山东代表团审议政府工作报告时，李克强总理向全国人大代表、浪潮集团董事长孙丕恕询问了浪潮在海外的发展情况，并勉励浪潮取得更大的发展。
  “你们在海外发展的怎么样？”李克强总理笑着问孙丕恕。
  “您2014年带我们去俄罗斯签约。”孙丕恕拿着一块服务器主板回答说，“这是我们在当地研发的服务器产品，现在已经开始规模化应用。”
  李克强接过主板仔细的看着，随后抬起头微笑着鼓励孙丕恕：“希望浪潮越做越大！祝你们取得更大的发展！”

  会议期间，在谈到山东要在推动新旧动能转换走在全国前面时，李克强总理说到：浪潮我去看过，他们发展大数据、云计算就是新动能！大众创业，万众创新不光是小微企业的事，大企业也要搞，而且大企业活力更大，浪潮与大中小企业一块合作，取得了很好的效果！
