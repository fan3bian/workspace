云计算中心出入管理注意事项制度

数据中心来访人员禁止携带下列物品入内 1. 手机、笔记本、平板电脑等电子设备。 2. 照相机、摄像机等录像设备及录音器材。 3. U盘、移动硬盘、光盘等存储设备和介质。 4. 食品、饮料、香烟。 5. 酒精及麻醉物品。 6. 易燃易爆物品。 7. 具有腐蚀性、挥发性的气体和液体。 8. 可干扰计算机设备和通信的电磁设备。 9. 放射性物品。 10. 安全管理人员提醒禁止携带的其他物品。 查看附件
