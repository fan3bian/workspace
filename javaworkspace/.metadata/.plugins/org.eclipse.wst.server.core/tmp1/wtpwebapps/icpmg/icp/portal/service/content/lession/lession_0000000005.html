云服务器的优势及应用场景

优势：

1、稳定性：服务可用性98%，数据可靠性99%，自动宕机迁移，数据备份和回滚，系统性能报警；

2、安全性：安全组规则保护，防DDoS系统，多用户隔离，防密码破解，多线BGP网络；
3、多线接入：基于BGP的最优路由算法；

4、弹性：10分钟内可启动或释放100台云服务器，在线不停机升级带宽，5分钟内停机升级CPU和内存；


应用场景：


在云服务器上安装各种应用软件，譬如Apache/Nginx、企业ERP/CRM管理软件、游戏软件、教育软件等；负载均衡服务SLB和多个云服务器连接成应用服务集群, 后端数据库采用高性能的关系型数据库服务RDS，此结构下应用系统的服务能力可以水平扩展；云服务器与开放存储服务OSS, 开放结构化数据服务OTS结合使用以提供更大的存储和访问能力。
