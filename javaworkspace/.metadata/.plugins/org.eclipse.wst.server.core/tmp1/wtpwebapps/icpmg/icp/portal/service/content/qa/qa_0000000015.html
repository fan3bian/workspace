什么是自动快照？自动快照有什么作用？

快照是磁盘上的数据在某一个时间点的拷贝；由用户设置策略，并自动为用户创建可管理的快照称作自动快照。支持快照回滚、快照删除、创建自定义镜像、通过快照购买磁盘等操作，自动快照可以让用户在不手动干预的情况下，根据预先设置的规则，周期性地为磁盘创建快照，以最大限度保证用户数据正确性和安全性。

您可以在ECS控制台--【全部快照】--右上角【 自动快照设置】进行管理自动快照。
