浪潮卫计委数据中心解决方案

方案概述

随着以计算机和网络通信为核心的信息化技术的发展飞速发展，信息化浪潮势不可挡，迅速地延伸到人类生活和社会的各个方面，也不可避免地改变着医疗技术和医疗模式。信息技术与医疗的结合已经成为当今世界医疗改革和发展的有机组成部分。浪潮提供的云数据中心核心设备助力卫计委数据中心搭建工作，实现了卫计委云计算中心平台的建设与升级改造，完成了远程平台软件硬件的建设，为卫计委在数据中心的建设中提供了完善的解决方案。

方案架构

浪潮数据中心基础设施产品的设计理念是：功能集中化，设计模块化，管理自动化。包括SmartRack服务器整机柜解决方案、模块化数据中心、云海Smartcloud集装箱数据中心等产品。能够为您提供完善的数据中心解决方案。

浪潮通过在医疗行业多年丰富的经验，对卫计委单位的数据中心建设有很多成熟的经验，我们提出针对卫计委各个业务系统、数据存储系统、数据保护系统、供电系统等大型数据中心的建设上提供了自己的建设模式，如下图：

浪潮卫计委数据中心建设解决方案涉及的业务系统包括：电子健康档案、电子病历库、影像库、资源库、与其它体系系统联通、基础云平台功能，远程医疗平台的完善与建设。

具体建设内容为：

（1）云计算中心平台的建设与升级改造：是指在原有的国家卫生部试点的容量基础上，主要从硬件设备上的建设。

（2）远程平台软件硬件的建设：是指在国家级远程医疗健康服务平台接入能力上的扩建工作。

（3）社会公众健康云服务平台：重点是针对门户方面的建设，包括相应的Internet门户。

（4）血液云管理、监督、基药行业等应用软件体系建设：是指基于云计算平台的有关血液、监督、基药等应用软件体系的建设。

（5）卫生行业管理及统计分析决策服务平台等内容，均是指基于云计算平台的管理、监督与分析决策应用软件体系建设。

（6）电子健康档案、电子病历库、影像库、资源库的扩建（含建档率）和资源数据库建设以及基础数据库的建设。

（7）平台与其它行业信息平台系统的连通（如：气象、公安、交通等体系）。

（8）卫计委平台网络、安全设备上的建设。

（9）卫计委数据中心的UPS电源系统建设。

方案产品介绍

AS500系列、AS1000系列

浪潮在线集中存储设备，具备领先的可靠性、可扩展性和易管理、易维护特性；支持数据快照、数据复制等功能，更支持远程数据同步的高端应用，是大型信息系统建设中构建数据存储平台的最优之选。

方案价值

　通过提供高性能的硬件环境，为用户提供可靠、可用的数据中心，实现如下价值：（1）政府通过政策支持，达到缓解老百姓“看病贵、看病难”难题，办大事的目的；（2）提高全国医疗服务能力，吸引患者在当地看病，同时通过新增服务项目，增加收益，也省去了建立心专业仪器和配备专职检查人员，降低经营成本；（3）大型医院和区县医院可扩大区域学术影响，能将更多精力花在解决疑难重症；（4）老百姓既可免除旅途劳顿、住宿等不必要的支出，节省时间，又可在家乡得到上级医院专业、高水平的医疗服务。
