食药监“智慧监管”平台

方案概述

浪潮现拥有面向食品、药品、保健食品、化妆品、医疗器械的各类业务监管系统，涵盖食药监的全业务范围，包括行政执法、信息监测、应急管理、公共服务、内部管理、决策支持六大业务平台，实现了上述“四品一械”从生产--流通—消费过程中的全生命周期的安全监管。

方案架构


浪潮软件食药监智慧监管平台的总体架构是基于社会化协作与共享的思想提出，采用云计算、大数据处理、智能WEB、智能消息推送、社会化协作、服务化架构等关键技术，通过不断扩大开放政府（企业）信息和服务资源，调动社会力量，依托政府（企业）信息和服务资源，开放安全可控的Web API服务，开发更多更好的应用，打造全新的松耦合、高效和高可用的信息平台，创造政府、企业与社会共同参与的信息化建设新模式，为信息化建设和发展带来新的活力。


方案价值

食药监“智慧监管”平台打造全新的松耦合、高效和高可用的信息平台，创造政府、企业与社会共同参与的信息化建设新模式，为信息化建设和发展带来新的活力。
