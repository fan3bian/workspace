产品详情Product details

  基于网络的云盘的出现极大地改善了信息存储和利用方面存在的难题。通过集群应用、网格技术或分布式文件系统等功能，将网络中的大量存储设备通过软件集合起来协同工作，共同对外提供一致的使用界面。相对于传统的存储方式，具有访问速度快、数据安全性保障高、管理维护方便、扩展性强、使用便捷和成本低廉等优点。

  云盘基于云存储技术，主要面向政府部门、大中型企事业单位、学校等用户，提供数据存储、知识管理、内容协作共享的专业解决方案。

功能描述Functional description

多种接入方式：

  • 网页版
  • 客户端版
  • 移动设备版

内容共享与协作：

  • 具有多用户协作模式，通过组织机构的权限管理和控制，可以将要协作的文档在多个用户或部门间共享。利用内容共享功能，用户可以将指定的文档分享给指定的用户

支持多种下载：

  • 单个文件的下载
  • 单个文件夹的打包下载
  • 多个文件的批量打包下载

文件加密存储：

  • 提供内容加密功能以保证数据安全（会增加服务器系统性能）

优化文件存储：

  • 对于用户上传的文件，可以解决两个完全相同的文件重复存储的问题，可有效的节省存储空间

文档格式转换：

  • 为了便于用户在浏览器中查看文档，浪潮云网盘提供在线文档格式转换、图片压缩裁剪等功能。浪潮云网盘支持以下格式的在线浏览：文档：doc、docx、xls、xlsx、ppt、pptx、txt、wps、pdf 等；图片：jpg、jpeg、png、gif、bmp 等；视频：avi、rmvb、mp4、flv 等。

空间配额管理：

  • 空间配额管理功能让管理员来维护用户所能使用的最大空间。通过弹性的配额管理可以灵活的控制和分配可用空间，提高存储资源的利用效率。
