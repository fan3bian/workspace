listZones

获取zone

请求参数


名称

类型

是否必须

描述

available

bool

是

若想获取所有可用的zone,则为true;如果想获取至少包含一个VM的zone,则为false

domainid

string

否

与zone相关联的domainid

id

string

否

zone id

keyword

List<string>

否

 

name

string

否

zone name

networktype

string

否

 

page

 

 

 

pagesize

 

 

 

showcapacities

bool

否

是否返回zone中资源利用情况


响应参数


名称

描述

id

Zone id

affinitygroupid

the UUID of the affinity group associated, null for public zones

allocationstate

the allocation state of the cluster

description

Zone description

dhcpprovider

the dhcp Provider for the Zone

displaytext

the display text of the zone

dns1

the first DNS for the Zone

dns2

the second DNS for the Zone

domain

Network domain name for the networks in the zone

domainid

the UUID of the containing domain, null for public zones

domainname

the name of the containing domain, null for public zones

guestcidraddress

the guest CIDR address for the Zone

internaldns1

the first internal DNS for the Zone

internaldns2

the second internal DNS for the Zone

ip6dns1

the first IPv6 DNS for the Zone

ip6dns2

the second IPv6 DNS for the Zone

localstorageenabled

true if local storage offering enabled, false otherwise

name

Zone name

networktype

the network type of the zone; can be Basic or Advanced

securitygroupsenabled

true if security groups support is enabled, false otherwise

vlan

the vlan range of the zone

zonetoken

Zone Token

capacity(*)

the capacity of the Zone

capacitytotal

the total capacity available

capacityused

the capacity currently in use

clustered

the Cluster ID

clustername

the Cluster name

percentuserd

the percentage of capacity currently in use

podid

the Pod ID

podname

the Pod name

 type

the capacity type

 zoneid

the Zone ID

 zonename

the Zone name
