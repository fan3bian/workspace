产品详情Product details

  为各行业用户数据中心提供本地或异地的容灾服务，包括容灾的建设与托管以及在线数据级容灾服务。云容灾产品主要是在浪潮云计算中心为用户建立远程的容灾系统，该系统是用户本地关键应用数据的一个可用副本。在用户本地数据或应用系统出现灾难时，远程保存有一份可用的关键业务的数据副本可快速提供灾难恢复。

  在云容灾服务的帮助下，用户不用再担心如何防止数据的丢失、如何保证数据的安全、以及需要提前采购多少用于数据容灾的存储空间等相关问题，把更多的精力聚焦在如何利用数据来创新。

  浪潮云容灾在2014年摘得国内外云服务多个奖项

  • 2014第二届全球云计算大会—“云鼎奖”—全球创新产品奖
  • 2014 CITE（第二届中国电子信息博览会）创新产品应用奖
  • 多个文件的批量打包下载
  • 第十八届中国国际软件博览会创新奖
