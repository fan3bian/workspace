『朝闻天下』国家出台鼓励云计算发展意见、政府购买云服务IT企业忙转型
1月17日，央视朝闻天下播出《国家出台鼓励云计算发展意见》、《政府购买云服务IT企业忙转型》。目前，我国云计算在发展过程中遇到了一些问题，比如像数据能力薄弱、还有数据资源开放共享不够、数据中心无序发展等。昨天，国务院印发了《关于促进云计算创新发展培育信息产业新业态的意见》，这个意见将引导云计算健康发展。

【新浪财经】浪潮云服务再获可信云服务认证
1月29日，数据中心联盟会员大会在京召开。本次会议正式发布了第三批通过可信云服务认证的云服务名单。作为中国领先的云计算整体解决方案供应商和云服务商，浪潮继2014年7月云服务器获首批可信云服务认证后，在线应用舆情分析云服务再次顺利通过认证，获得权威认可。
