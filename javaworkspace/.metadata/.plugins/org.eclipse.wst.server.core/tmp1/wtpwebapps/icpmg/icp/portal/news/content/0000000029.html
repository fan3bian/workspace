浪潮中标国家食药监总局云服务建设项目

 日前，浪潮成功中标国家食品药品监督管理总局食品许可云服务建设项目。该项目的中标标志着浪潮实现了在部委总部云服务业务的又一突破，并为后续食药监行业市场的开拓打下了坚实的基础。

 国家食品药品监督管理总局，是中华人民共和国国务院正部级直属机构，成立于2013年3月22日，取代了原国家食品药品监督管理局和国务院食品安全委员会办公室。国家食品药品监督管理总局食品经营许可数据库项目系统，将为总局及省分机构业务系统的建设提供基础资源服务，以达到简化项目建设流程、节约建设成本的目的。该项目采用购买浪潮云服务的模式进行建设运营，系统整体依托浪潮云中心及平台，功能先进、覆盖广泛、安全实用、监管有效、资源共享。浪潮按照信息安全等级保护三级标准，为食药总局提供专有云服务，本期项目浪潮在北京建设生产中心和灾备中心两个云中心，支撑业务系统稳定、安全、高效、可靠运行；同时建立运维服务保障体系，确保全国大集中应用可用性达到99.99%。系统建成后，生产中心部署核心业务应用系统，灾备中心实现对生产中心关键数据的灾备功能。

 2015年初开始，国家食品药品监督管理总局食品许可云服务建设项目正式启动。由于云服务领域竞争激烈，国内各大云服务厂商都看好该项目的后续市场前景，均要势在必得。经过激烈的角逐，浪潮在云计算、大数据等方面的强大实力和丰富经验得到客户方的高度认可，也有效促使客户方信息化项目从传统的采购方式向购买云服务模式的转变。浪潮云服务最终以超过用户预期的方案及服务打败竞争对手。该项目的成功中标，再次证明：浪潮，作为行业级云服务领导者，自主可靠，安全可信，值得用户信赖。

 在行业云领域，浪潮云服务将不断前行，永不止步！
