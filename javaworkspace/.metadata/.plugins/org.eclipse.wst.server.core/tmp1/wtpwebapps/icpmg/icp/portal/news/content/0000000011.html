浪潮云服务中标阜阳市政务云项目

近日，浪潮成功中标阜阳市政务云服务项目。这是阜阳智慧城市进程中的关键一步，标志着浪潮与阜阳市的云服务合作正式进入实质性落地应用阶段，并为后续阜阳市云服务业务的深入开展奠定了坚实的基础。

阜阳政务云项目采用公开招标方式，明确服务目录及服务单价，实现了政府信息化建设弹性计价的模式创新。同时，浪潮通过与本地运营商合作，快速搭建云服务平台，向阜阳市委市政府输出云服务。本次主要针对阜阳市基于云计算的公共信息服务平台开展合作。实现了政务云业务的快速部署与交付。

此次中标，是浪潮云服务在政务云上的又一次成功实践，进一步验证了浪潮“灵活定制、随需而动”的创新型行业级云服务模式，也标志着安徽市场对浪潮云服务整体能力的认可，鼓励我们在前进的道路上再创佳绩！
