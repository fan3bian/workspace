浪潮政务服务中心综合解决方案

方案概述

政务服务中心是我国行政审批制度改革进程中出现的新型机构，其遵循高效便民的服务精神，以整合、协作、集中的运作模式，依靠先进的服务手段，承担着代表政府部门集中办理行政审批业务和部分公共服务事项的职责。 浪潮政务服务中心综合解决方案围绕服务、业务、管理三个方面，以信息化建设为抓手，提升综合服务能力  建立统一、开放、有活力、用户可参与的网上办事大厅  建立梳理、运行、公开、监督监察统一管理体系  建设智能化服务大厅行政管理平台  建立各级政务服务的统一信息资源共享和数据交换标准  建立统一工具化、定制化、易用化的系统建设规范，形成业务和信息化建设无缝结合

方案架构


统一门户

将内部建设的各个应用系统进行界面整合，使用户能够在一个界面上完成所有基础工作，而不需要关心使用哪套应用系统。

行政管理系统

对进驻政务大厅的部门、窗口、人员、部门事项和政务办内部人员、资产和进驻部门与政务办日常业务往来以及管理评价等进行全方位的综合管理。

数字大厅

数字大厅以已有设备为基础，通过信息化的集成建设，构建数据互通、信息互联的虚拟数字大厅，实现办事群众大厅办事的便捷化、内部管理智能化、考评精细化、监控自动化。

通用审批系统

为进驻政务服务中心，但又没有自己的业务审批系统的委办局提供公用的业务审批平台，实现收件、补齐补正、挂起、业务流转及领证等操作以及表单、流程、打印表单、证照等个性化业务操作。

并联审批系统

以大幅度减少审批事项、大幅度缩短审批时限为目标，通过“整合流程、一门受理、并联审批、信息共享、限时办结”等改革方式，明确各方责任，强化监督机制。适用于建设工程项目并联审批、企业设立等需要多部门联合协作的业务领域。

接入与集成平台

在各委办局已有审批系统的基础上，建设统一的行政审批事项管理，并将现有行政许可审批事项、非行政许可审批事项、备案事项的业务数据进行统一管理及监督，为统一的网上办理、并联审批、多级业务联动以及行政审批服务轨迹提供数据和应用基础。

电子证照系统

建立证照电文数据共享平台，将电子证照与电子批文应用到行政审批流程中。企业、民众申报行政审批事项无需提供前置证照的纸质材料，而由受理部门窗口通过电子证照共享平台查验。审批生成的证照自动存储到电子证照库中，供以后查验，将证照电子信息化提升到为电子政务公共服务的高度。

网上办事大厅

以“实体服务大厅业务上网，网上办事大厅服务落地”为基本要求，充分利用现有电子政务建设成果，整合已有资源，以公众为导向，精心规划建设网上办事大厅，实现以市民为中心、以服务化管理为目标的高效电子政务平台，进一步整合提升公共服务能力，提供“一站式、一网式”的统一政务服务窗口。



方案价值

利用统一的信息交换平台，在各个政府部门业务系统之间彼此相互独立的情况下，通过消息传递的松耦合方式，将不同平台、不同架构和不同功能的业务系统有机地连接起来，实现各政府部门间的信息交换。建立统一、开放、有活力、用户可参与的网上办事大厅；建立梳理、运行、公开、监督监察统一管理体系；建设智能化服务大厅行政管理平台；建立各级政务服务的统一信息资源共享和数据交换标准；建立统一工具化、定制化、易用化的系统建设规范，形成业务和信息化建设无缝结合。
