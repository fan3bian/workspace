山东盐业集团管理系统

公司概述

山东盐业集团的HR系统、财务系统、国资监管系统及资金系统在浪潮济南云计算中心成功上线落地。系统将服务于盐业集团在全省17个市、124个县（市、区）垂直管理的盐业经营管理机构（盐业公司、盐务局），以及6个直属食盐定点生产企业，1个盐业转运处，其销售网络遍布全省城乡，年度商品销售收入在14～18亿元，利税1.5～2亿元，年产各类食盐120万吨。山东盐业集团的HR系统、财务系统、国资监管系统及资金系统在浪潮济南云计算中心成功上线落地。
