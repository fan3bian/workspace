浪潮天梭K1-950落户常德 开启智慧城市新篇章

浪潮天梭K1-950成功落户常德云计算中心，保障了湖南省常德市云计算中心数据库安全高效的运行，满足了客户对云计算、大数据的多样化需求，实现了浪潮在湖南省政务云关键应用领域的突破！

2014年4月，浪潮集团与常德市委市政府签定了智慧常德的战略合作协议。常德市政府与浪潮集团联手合作，依托现有基础，通过“建设云中心、整合数据资源、搭建公共服务平台”三步曲搭建智慧常德整体框架，整合政府传统组织内部的数据，整合政府跨部门之间的数据，创新运营模式，有序对外开放，并通过数据开放能够实现数据的深加工，扎实推进公共服务的应用，带动一批中小微企业，培育新的信息消费产业链条。它开创了智慧城市建设的新模式，通过政府公开招标选择智慧城市建设的战略合作伙伴；云中心的概念由传统狭义的Iaas层平台云计算中心上升到了集Iaas、Paas、Saas三层平台于一体的广义的云计算中心；服务模式由传统的基础设施服务改变为集软硬一体全方位运营；项目承建由传统采购建设转变为投资运营。

云计算作为战略性新兴产业，是提升政府公共管理和公共服务水平，带动地方产业结构调整与升级的关键手段。随着我国经济的快速发展和信息化建设的推进，国内各行业云服务建设也会在国家社会全局中扮演越来越重要的角色，2014年，浪潮天梭K1完成了金融、电力、公安、交通等12个行业市场的应用突破，建设银行、农业部、胜利油田、北京市财政局、广州白云机场、洛阳银行等都在核心业务中用天梭K1替代了进口产品。目前，浪潮天梭K1在国内已经获得广泛应用，形成了产业规模。浪潮将自主化进程向更深层次推进，助力打造中国云计算生态链。
