【新浪财经】浪潮云服务再获可信云服务认证

1月29日，数据中心联盟会员大会在京召开。本次会议正式发布了第三批通过可信云服务认证的云服务名单。作为中国领先的云计算整体解决方案供应商和云服务商，浪潮继2014年7月云服务器获首批可信云服务认证后，在线应用舆情分析云服务再次顺利通过认证，获得权威认可。

高标准、严要求的可信云服务认证

“可信云服务工作组” 由工信部电信研究院牵头成立，大量借鉴了国外先进经验和国内云计算企业的实践经验，制定了《云计算服务协议参考框架》、《可信云服务认证评估方法》、《可信云服务认证评估操作办法》三个标准，内容包括数据私密性、数据知情权、故障恢复能力、网络接入性能等16个指标。其高规格、高标准对于国内云服务提出了更高的要求，且对云市场的运营体系规范和行业规模扩大都有着重要的积极作用。截至目前为止，共有67个云服务通过了可信云服务认证，分别由32个服务商提供，通过率为64%。

浪潮舆情云监测系统服务，是利用爬虫及搜索引擎技术对互联网的标题、网页内容、论坛、BBS、贴吧、微博、作者、时间、出处等各个方面及多个维度进行数据采集，然后通过基于自主知识产权的专利算法进行深度的数据挖掘以及智能化的数据分析，从而生成 Word、Web、PDF 等各种形式的报表，这被称作舆情简报服务。浪潮舆情云监测系统主要包括舆情信息采集，舆情信息处理，舆情图表分析，多维度微博监测，舆情危机预警，舆情统计报告等功能。浪潮舆情云监测系统可为企业用户提供各类信息，包括但不限于政策、行业动向、友商动态、负面新闻，从而为企业用户提供各类决策依据，有助于企业用户加强品牌建设。同时舆情简报服务可为政府用户提供舆论热点分析、民意搜索与调查、负面舆论的搜集、消除及引导等功能。

在评估过程中，浪潮云服务多项服务标准在业界表现出众，赢得了业内的认可，并通过了市场的检验。

自主可靠、安全可信的浪潮云服务

作为国内云服务领域不容忽视的一股力量，“铁三角”是浪潮行走行业云市场的独特的优势。据浪潮云服务总经理颜亮介绍，“铁三角”代表着浪潮集团的三方面能力：浪潮软硬件产品全覆盖，具备作为产品供应商的能力；浪潮具有国内特一级计算机系统集成资质，具备作为解决方案供应商的能力；跨区域 IDC/ISP 牌照的获批让浪潮具备作为运营服务供应商的能力。“铁三角”的优势让浪潮成为全国唯一一家具备整体交付能力的云服务商。

同时，浪潮从关键应用主机、X86服务器、海量存储、第四代数据中心，到云服务管理平台、大数据平台均实现自主研发，能为用户提供自主可靠、安全可信的云服务，是最适合中国国情的云服务模式。

2014年，浪潮全面践行云战略转型，凭借自主可靠的软硬件一体化能力和实践，已经发展成为行业级云服务领导者，实现了政务云市场占有率第一。目前浪潮云服务已经在全国49个城市和地区提供云服务，随着服务能力与产品线的完善，陆续得到国内外权威机构、云服务产业与广大用户的普遍认可信赖。
